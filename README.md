
# 🚀 StartupForge - Client

StartupForge is a modern Startup Team Builder platform where startup founders can create startup profiles, post opportunities, and recruit talented collaborators. Developers, designers, marketers, and other professionals can explore opportunities and apply to join startup teams.

## ✨ Features

- 🔐 Better Auth Authentication
- 🔑 Google & Credentials Login
- 👥 Role Based Access Control (Admin, Founder, Collaborator)
- 🏢 Startup Management
- 💼 Opportunity Management
- 📝 Opportunity Application System
- 👤 User Profile Management
- 💳 Stripe Premium Subscription
- 📊 Dashboard Overview
- 🚫 Admin Block/Unblock Users
- ✅ Admin Approve/Reject Startups
- 🔍 Search Opportunities
- 🎯 Filter Opportunities
- 📱 Fully Responsive UI
- 🎨 Modern UI with Tailwind CSS

## 🛠️ Technologies Used

- Next.js 16
- React 19
- Tailwind CSS
- Axios
- Better Auth
- TanStack Query
- React Hook Form
- Stripe
- React Icons
